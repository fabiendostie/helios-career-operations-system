Module Description

Main resilient LLM client implementation.

Classes

ResilientLLMClient

Resilient LLM client with caching, retries, fallbacks, and circuit breakers. This client provides a robust interface to LLM services with: - Redis caching for identical prompts - Automatic retries with exponential backoff - Fallback between multiple providers - Circuit breaker pattern to prevent cascading failures - Comprehensive logging and monitoring
