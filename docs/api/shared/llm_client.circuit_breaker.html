Module Description

Circuit breaker implementation for LLM client resilience.

Classes

States of the circuit breaker.

CircuitBreakerStats

CircuitBreaker

Circuit breaker implementation to prevent cascading failures. The circuit breaker has three states: - CLOSED: Normal operation, requests are allowed - OPEN: Requests are rejected immediately due to failures - HALF_OPEN: Testing state, limited requests allowed
