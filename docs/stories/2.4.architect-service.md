# Story 3.1: ARCHITECT Agent Service

## Status: ✅ COMPLETED - Fully Verified and Operational

## Story
**As a** career professional using Helios,
**I want** an intelligent agent that generates professional, ATS-compliant documents using my career analysis,
**so that** I can receive high-quality resumes and cover letters tailored to specific opportunities and formatted for maximum application success rates.

## Acceptance Criteria

1. The service must implement ATS-compliant document generation using "T-Shaped" resume architecture with Impact Summary and Experience Log sections
2. The system must support multiple template formats (2-3 variations) with single-column layouts and standard fonts
3. The service must generate multiple output formats: Markdown, PDF, and optionally DOCX for maximum compatibility
4. The system must dynamically select and customize content based on ANALYST recommendations and career path insights from STRATEGIST
5. **CRITICAL: The service must implement a Sophisticated Dynamic Research Engine using semantic NLP analysis (spaCy + Transformers) with Bronze/Silver/Gold data architecture, Prefect orchestration, and authoritative sources (McKinsey, Gartner, ATS vendors, BLS API) to provide genuine intelligence instead of keyword matching**
6. The service must implement template customization based on career path, industry, and job-specific requirements using live market data
7. The service must integrate with Story 2.1 Orchestrator for session management and the `/build` command
8. The system must maintain >85% test coverage following coding standards and respond within <30s per document
9. The service must generate cover letters using the "Pain & Promise" model with company-specific customization using real-time company research
10. **The Dynamic Research Engine must implement Medallion Architecture (Bronze/Silver/Gold layers) with PostgreSQL persistence, scheduled Prefect workflows for weekly intelligence updates, semantic NLP processing pipeline, and multi-source data ingestion from authoritative sources with quality scoring and confidence metrics**

### Sophisticated Dynamic Research Engine Architecture

**CRITICAL UPDATE**: The ARCHITECT service now implements a production-grade research intelligence system that replaces shallow keyword-based research with deep semantic analysis.

**Key Architecture Changes**:
- **Medallion Data Architecture**: Bronze (raw), Silver (processed), Gold (analytics) layers with PostgreSQL persistence

- **Authoritative Data Sources**: McKinsey Digital, Gartner HR, ATS vendor documentation, BLS API, Reddit recruiting forums
- **Automated Orchestration**: Prefect workflows for weekly scheduled research updates with error handling
- **Quality Assurance**: Confidence scoring, data freshness validation, multi-source verification

**Research Intelligence Capabilities**:
1. **Industry Trends**: Semantic analysis of consulting reports (McKinsey, Gartner) for genuine market intelligence

4. **Company Intelligence**: Multi-source research for Pain & Promise cover letter customization
5. **Hiring Sentiment**: NLP analysis of recruiting forums for hiring manager preferences and frustrations

**Data Quality Standards**:
- Confidence scoring (0-1) for all intelligence with source attribution
- Maximum data age: 24 hours for industry trends, 12 hours for ATS compliance
- Multi-source validation with fallback intelligence for reliability
- Comprehensive error handling and graceful degradation

### ARCHITECT Agent Knowledge Integration
[Source: knowledge-base/agent-knowledge/Knowledge_Document_4_ARCHITECT.md]

**T-Shaped Resume Architecture**:
- **Top Bar (Impact Summary)**: 3-4 line dense summary answering: Who are you? What do you do? Key quantified achievement? Top 3-4 skills?
- **Vertical Bar (Experience Log)**: Reverse-chronological deep dive with bullet points proving summary claims

**ATS Compliance Standards (2025)**:
```python
ATS_COMPLIANCE_RULES = {
    "file_format": "PDF generated from text source (no complex .docx metadata)",
    "layout": "Strictly single-column (no multi-column layouts)",
    "fonts": "Standard sans-serif only (Arial, Calibri, Helvetica), 10-12pt",
    "graphics": "Prohibited - no images, icons, or skill bars",
    "keywords": "Critical keywords in relevant contexts (not keyword stuffing)",
    "headings": "Standard, simple headings (Professional Experience, Education, Skills)"
}
```

**Impact Summary Formula**:
```
[Adjective] [Target Role Title] with [X] years of experience specializing in
[Core Competency 1], [Core Competency 2], and [Tool/Technology]. Proven ability to
[Top Quantified Achievement] by leveraging expertise in [Skill Area].
```

**Cover Letter "Pain & Promise" Model**:
1. **Paragraph 1 (Hook & Pain)**: Address company directly, demonstrate understanding of their challenge
2. **Paragraph 2 (Bridge & Promise)**: Connect quantified achievement to their pain point with proof
3. **Paragraph 3 (Close & Call to Action)**: Reiterate excitement with confident call to action

# Document generation with retry logic
async def generate_document_with_retry(
    session_data: dict,
    recommendations: AnalysisRecommendations,
    request: ResumeGenerationRequest,
    max_retries: int = 2
) -> DocumentResponse:
    """
    Generate document with retry logic for transient failures

    Error Recovery:
    - PDF generation failure: Retry with different rendering engine
    - Template processing error: Fall back to simpler template
    - Memory error: Reduce concurrent operations and retry
    """
    last_exception = None

    for attempt in range(max_retries + 1):
        try:
            return await _generate_document_internal(session_data, recommendations, request)

        except MemoryError as e:
            logger.warning(f"Memory error on attempt {attempt + 1}: {e}")
            if attempt < max_retries:
                # Clear caches and retry with reduced memory footprint
                await clear_template_cache()
                await asyncio.sleep(1)  # Brief delay
                continue
            raise HTTPException(status_code=507, detail="Insufficient memory for document generation")

        except TemplateRenderError as e:
            logger.warning(f"Template rendering failed on attempt {attempt + 1}: {e}")
            if attempt < max_retries:
                # Fall back to simpler template
                request.template_preference = "minimal"
                continue
            raise HTTPException(status_code=422, detail="Unable to render document with current data")

        except Exception as e:
            last_exception = e
            if attempt < max_retries:
                logger.warning(f"Generation attempt {attempt + 1} failed: {e}, retrying...")
                await asyncio.sleep(0.5 * (attempt + 1))  # Exponential backoff
                continue

    # All retries exhausted
    logger.error(f"Document generation failed after {max_retries + 1} attempts: {last_exception}")
    raise HTTPException(status_code=500, detail="Document generation failed")

class PerformanceCircuitBreaker:

        self.failure_threshold = failure_threshold
        self.timeout = timeout
        self.failure_count = 0
        self.last_failure_time = None

# Memory-efficient document processing patterns
import asyncio
import gc
import weakref
from functools import lru_cache
from typing import Dict, Any, Optional
import tempfile
import os
from contextlib import asynccontextmanager

class MemoryOptimizedTemplateEngine:
    """Template engine with memory optimization patterns"""

    def __init__(self, max_cache_size: int = 50):
        self.max_cache_size = max_cache_size
        self._template_cache = weakref.WeakValueDictionary()
        self._template_access_count = {}

    @lru_cache(maxsize=50)  # LRU cache for compiled templates
    def get_compiled_template(self, template_path: str):
        """Get compiled template with LRU caching"""
        try:
            with open(template_path, 'r', encoding='utf-8') as f:
                template_content = f.read()

            # Use Jinja2 with memory-efficient settings
            from jinja2 import Environment, FileSystemLoader
            env = Environment(
                loader=FileSystemLoader(os.path.dirname(template_path)),
                auto_reload=False,  # Disable auto-reload for production
                cache_size=0,       # Disable Jinja2's internal cache (we handle caching)
                optimized=True,     # Enable template optimization
                finalize=lambda x: x if x is not None else ''  # Handle None values
            )

            return env.from_string(template_content)

        except Exception as e:
            logger.error(f"Failed to compile template {template_path}: {e}")
            raise TemplateRenderError(f"Template compilation failed: {e}")

    async def render_template_with_memory_management(
        self,
        template_path: str,
        context: Dict[str, Any],
        enable_streaming: bool = True
    ) -> str:
        """
        Render template with memory optimization strategies

        Memory optimization techniques:
        1. Streaming for large documents
        2. Context data chunking
        3. Aggressive garbage collection
        4. Temporary file usage for large renders
        """
        # Pre-render memory check
        initial_memory = psutil.Process().memory_info().rss / 1024 / 1024

        try:
            template = self.get_compiled_template(template_path)

            # For large contexts, use streaming approach
            if self._estimate_context_size(context) > 50:  # >50MB estimated
                return await self._render_with_streaming(template, context)
            else:
                return await self._render_standard(template, context)

        except MemoryError:
            # Emergency memory cleanup
            await self._emergency_memory_cleanup()
            logger.warning("Memory error during template rendering, attempting recovery")

            # Retry with minimal memory footprint
            return await self._render_minimal_memory(template_path, context)

        finally:
            # Post-render cleanup
            final_memory = psutil.Process().memory_info().rss / 1024 / 1024
            memory_delta = final_memory - initial_memory

            if memory_delta > 100:  # >100MB increase
                await asyncio.sleep(0.1)  # Brief pause for GC
                gc.collect()  # Force garbage collection

    def _estimate_context_size(self, context: Dict[str, Any]) -> int:
        """Estimate context size in MB"""
        try:
            # Rough estimation based on string lengths and object counts
            size_estimate = 0

            for key, value in context.items():
                if isinstance(value, str):
                    size_estimate += len(value.encode('utf-8'))
                elif isinstance(value, list):
                    size_estimate += len(value) * 1000  # Rough estimate
                elif isinstance(value, dict):
                    size_estimate += len(str(value))

            return size_estimate / (1024 * 1024)  # Convert to MB
        except:
            return 10  # Default conservative estimate

    async def _render_with_streaming(
        self,
        template,
        context: Dict[str, Any]
    ) -> str:
        """Render large templates using streaming approach"""
        # Use temporary file for large renders
        with tempfile.NamedTemporaryFile(mode='w+', delete=False, suffix='.html') as tmp_file:
            try:
                # Render in chunks to manage memory
                rendered_chunks = []

                # Split context into smaller chunks if possible
                if 'work_experience' in context and len(context['work_experience']) > 10:
                    # Process experience entries in batches
                    experience_batches = [
                        context['work_experience'][i:i+5]
                        for i in range(0, len(context['work_experience']), 5)
                    ]

                    for batch in experience_batches:
                        batch_context = context.copy()
                        batch_context['work_experience'] = batch

                        chunk = await asyncio.to_thread(template.render, batch_context)
                        rendered_chunks.append(chunk)

                        # Memory cleanup after each batch
                        del chunk
                        gc.collect()

                    # Combine chunks
                    full_render = '\n'.join(rendered_chunks)
                else:
                    # Standard render for smaller contexts
                    full_render = await asyncio.to_thread(template.render, context)

                # Write to temporary file first (memory management)
                tmp_file.write(full_render)
                tmp_file.flush()

                # Read back (allows OS to manage memory more efficiently)
                tmp_file.seek(0)
                final_content = tmp_file.read()

                return final_content

            finally:
                # Cleanup temporary file
                try:
                    os.unlink(tmp_file.name)
                except:
                    pass

    async def _render_standard(self, template, context: Dict[str, Any]) -> str:
        """Standard rendering with memory monitoring"""
        return await asyncio.to_thread(template.render, context)

    async def _render_minimal_memory(
        self,
        template_path: str,
        context: Dict[str, Any]
    ) -> str:
        """Emergency minimal memory rendering"""
        # Clear all caches
        self.get_compiled_template.cache_clear()
        gc.collect()

        # Use minimal context
        minimal_context = {
            'candidate_name': context.get('candidate_name', 'Professional'),
            'email': context.get('email', ''),
            'phone': context.get('phone', ''),
            'work_experience': context.get('work_experience', [])[:3],  # Limit to 3 jobs
            'education': context.get('education', [])[:2],  # Limit to 2 education entries
            'core_skills': context.get('core_skills', [])[:10]  # Limit skills
        }

        template = self.get_compiled_template(template_path)
        return await asyncio.to_thread(template.render, minimal_context)

    async def _emergency_memory_cleanup(self):
        """Emergency memory cleanup procedures"""
        # Clear template cache
        self.get_compiled_template.cache_clear()
        self._template_cache.clear()

        # Force garbage collection
        gc.collect()

        # Brief pause to allow memory reclamation
        await asyncio.sleep(0.5)

        logger.warning("Emergency memory cleanup executed")

            )

                pdf_content = pdf_file.read()

            yield pdf_content

        except MemoryError:
            logger.error("Memory error during PDF generation")
            # Attempt recovery with minimal settings
            yield await self._generate_minimal_pdf(html_content)

        finally:
            # Cleanup temporary files
            for temp_file in [temp_html_file, temp_pdf_file]:
                if temp_file and os.path.exists(temp_file):
                    try:
                        os.unlink(temp_file)
                    except:
                        pass

            # Force cleanup
            gc.collect()

    async def _generate_minimal_pdf(self, html_content: str) -> bytes:
        """Generate PDF with minimal memory usage"""
        # Simplified HTML content to reduce memory usage
        simplified_html = self._simplify_html_content(html_content)

        with tempfile.NamedTemporaryFile(mode='w', suffix='.html') as html_file:
            html_file.write(simplified_html)
            html_file.flush()

            import weasyprint
            html_doc = weasyprint.HTML(filename=html_file.name)
            return html_doc.write_pdf()

    def _simplify_html_content(self, html_content: str) -> str:
        """Simplify HTML content to reduce memory usage"""
        # Remove unnecessary elements, compress whitespace, etc.
        import re

        # Remove comments
        html_content = re.sub(r'<!--.*?-->', '', html_content, flags=re.DOTALL)

        # Compress whitespace
        html_content = re.sub(r'\s+', ' ', html_content)

        # Remove empty elements
        html_content = re.sub(r'<(\w+)[^>]*>\s*</\1>', '', html_content)

        return html_content.strip()
