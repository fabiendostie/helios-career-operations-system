## Story
**As a** career professional using Helios,
**I want** an intelligent agent that performs deep market analysis and resume optimization,
**so that** I can understand my competitive positioning and receive data-driven recommendations for career advancement.

## Acceptance Criteria

1. The service must implement a 6-step analysis pipeline: Resume Deconstruction → Market Correlation → ATS Simulation → Skill Recalibration → Career Path Inference → Final Report Generation
2. The system must perform NER extraction using spaCy with custom professional entities (SKILL, TOOL, METRIC, ACTION_VERB, RESPONSIBILITY)
3. The service must simulate market analysis using structured job and compensation databases with realistic data schemas
4. The system must generate ATS readiness scores using weighted criteria (keyword match 40%, format parsability 30%, quantification 20%, action verbs 10%)
5. The service must implement the Skill Framing Matrix for skill classification (Leverage/Upskill/Maintain/De-emphasize quadrants)
6. The service must integrate with Story 2.1 Orchestrator for session management and the `/analyze` command
7. The system must maintain >85% test coverage following coding standards
8. The service must respond within <5s for complete analysis pipeline execution

```python
    "bonus_usd": "integer"

```

**Step 3 - ATS Scoring Criteria**:
- Keyword Match Score (40%): Cosine similarity between resume and job description
- Format Parsability Score (30%): Based on NER extraction success rate
- Quantification Score (20%): Percentage of bullets containing METRIC entities
- Action Verb Score (10%): Percentage using high-impact action verbs

**High-Impact Action Verbs by Category**:
- Technical: Architected, Engineered, Deployed, Refactored, Automated
- Management: Orchestrated, Governed, Mentored, Scaled, Piloted
- Growth/Sales: Accelerated, Evangelized, Penetrated, Negotiated, Captured
- Analysis: Synthesized, Modeled, Forecasted, Quantified, Inferred

**Step 4 - Skill Framing Matrix**:
```
             High Market Demand    Low Market Demand
High Prof.   LEVERAGE (Quadrant I)  MAINTAIN (Quadrant III)  
Low Prof.    UPSKILL (Quadrant II)  DE-EMPHASIZE (Quadrant IV)
```
